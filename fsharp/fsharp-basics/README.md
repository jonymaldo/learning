# Fsharp basics

This project is used to learn basic f# language features

## Setup

1. `dotnet new xunit -lang f#` -> creates a test project with dotnet cli
2. `dotnet add package FsUnit.xUnit --version 3.0.0` -> add fsunit nuget to the project
